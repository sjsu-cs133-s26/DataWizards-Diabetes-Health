## Project Overview

This project builds a data engineering pipeline around a public health and
demographic dataset pertaining to diabetes research. We will ingest the raw
dataset, validate and clean fields, standardize formats (dates, categories,
text fields), and produce analytics-ready tables for querying and
visualization.

## Dataset

**Dataset Name:** Comprehensive Diabetes Clinical Dataset

**Domain:** Health

**Source (Kaggle):** [https://www.kaggle.com/datasets/priyamchoksi/100000-diabetes-clinical-dataset](https://www.kaggle.com/datasets/priyamchoksi/100000-diabetes-clinical-dataset) 

**Usage:** Public + free to use (per Kaggle dataset access)
