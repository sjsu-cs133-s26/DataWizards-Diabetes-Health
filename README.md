## Project Overview
This project builds a data engineering pipeline around a public Netflix catalog dataset. We will ingest the raw dataset, validate and clean fields, standardize formats (dates, categories, text fields), and produce analytics-ready tables for querying and visualization.

## Dataset
**Dataset Name:** Netflix Movies and TV Shows  
**Domain:** Entertainment / Streaming Media  
**Source (Kaggle):** https://www.kaggle.com/datasets/shivamb/netflix-shows  
**Usage:** Public + free to use (per Kaggle dataset access)
